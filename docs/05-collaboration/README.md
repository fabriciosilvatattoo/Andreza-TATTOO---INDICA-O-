# Colaboração Externa

Este diretório armazena o histórico de interações e decisões tomadas com colaboradores externos ao projeto, sejam eles humanos ou outros modelos de IA (como o Claude).

Manter esses registros é crucial para:
- Rastrear a origem de ideias e decisões de arquitetura.
- Manter um contexto compartilhado.
- Evitar a repetição de discussões.
